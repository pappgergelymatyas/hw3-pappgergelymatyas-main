# Task

Given an **n** integer

Try to return the following as a string:

123...n

Note that "..." represents the consecutive values in between.

# Example

**n** = 5

Return the string "12345".

# Input Format

An integer **n**.

# Constraints

1 <= **n** <= 150

# Output Format

Return the list of integers from 1 through **n** as a string, without spaces.

# Sample Input 0

    3
# Sample Output 0

    "123"
